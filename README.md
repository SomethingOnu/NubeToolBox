## 概要

「**Nube Tool Box**」とは、**おぬ氏が個人的に利用する機能やツール**を纏めたガジェットです。現在は**基本情報の表示**や**音楽の再生**、最小限の**ユーティリティー**、**文字列の操作**、**ソフトウェアの起動**が行えます。
